生产者示例
<?php use Tinywan\RedisStream\RedisStreamQueue; use Tinywan\RedisStream\MonologFactory; // 基础配置 $redisConfig = ['host' => '127.0.0.1', 'port' => 6379]; $queueConfig = [ 'stream_name' => 'my_queue', 'consumer_group' => 'my_group']; // 创建日志记录器 $logger = MonologFactory::createLogger('redis-stream-app'); // 创建队列实例 $queue = RedisStreamQueue::getInstance( $redisConfig, $queueConfig, $logger ); // 批量发送消息 $messages = [ ['type' => 'user_registered', 'user_id' => 123], ['type' => 'order_created', 'order_id' => 456], ['type' => 'payment_success', 'amount' => 99.99] ]; foreach ($messages as $message) { $messageId = $queue->send($message); echo "Sent: " . $messageId . "\\n"; }
消费者示例
<?php use Tinywan\RedisStream\RedisStreamQueue; use Tinywan\RedisStream\MonologFactory; // 基础配置 $redisConfig = ['host' => '127.0.0.1', 'port' => 6379]; $queueConfig = [ 'stream_name' => 'my_queue', 'consumer_group' => 'my_group']; // 创建日志记录器 $logger = MonologFactory::createLogger('redis-stream-app'); // 创建队列实例 $queue = RedisStreamQueue::getInstance( $redisConfig, $queueConfig, $logger ); // 消费消息 (回调方式) $message = $queue->consume(function($message) { echo "Processing: " . $message['type'] . "\\n"; // 业务逻辑处理 switch ($message['type']) { case 'user_registered': // 处理用户注册逻辑 break; case 'order_created': // 处理订单创建逻辑 break; } return true; // 自动确认 }); // 或者直接消费方式 // $message = $queue->consume(); // if ($message) { // // 处理消息 // $queue->ack($message['id']); // }

核心特性

⚡

超高性能

基于 Redis 5.0+ Stream 数据结构，性能卓越，支持高并发消息处理

🔄

多生产者/消费者

支持多个生产者和消费者同时工作，提供完整的消费者组管理

💾

消息持久化

可靠的消息持久化存储，确保数据不丢失，支持断点续传

✅

ACK 确认机制

完善的消息确认机制，保证消息可靠投递和处理

🔄

智能重试

内置消息重试机制，自动处理失败消息，提高系统可靠性

⏰

延时消息

支持延时消息和定时消息，提供灵活的时间控制策略
