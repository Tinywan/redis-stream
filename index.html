快速安装

快速开始

生产者示例
<?php // 基础配置 $redisConfig = ['host' => '127.0.0.1', 'port' => 6379]; $queueConfig = [ 'stream_name' => 'my_queue', 'consumer_group' => 'my_group' ]; // 创建队列实例 $queue = RedisStreamQueue::getInstance( $redisConfig, $queueConfig, $logger ); // 发送消息 $messageId = $queue->send([ 'type' => 'user_registered', 'user_id' => 123, 'email' => 'user@example.com' ]);
消费者示例
<?php // 创建消费者实例 $consumer = new Consumer($queue); // 消费消息 $message = $consumer->consume(function($message) { echo "Processing: " . $message['type'] . "\\n"; // 业务逻辑处理 // ... return true; // 成功处理 }); if ($message) { echo "Message processed: " . $message['id'] . "\\n"; }

核心特性

⚡

超高性能

基于 Redis 5.0+ Stream 数据结构，性能卓越，支持高并发消息处理

🔄

多生产者/消费者

支持多个生产者和消费者同时工作，提供完整的消费者组管理

💾

消息持久化

可靠的消息持久化存储，确保数据不丢失，支持断点续传

✅

ACK 确认机制

完善的消息确认机制，保证消息可靠投递和处理

🔄

智能重试

内置消息重试机制，自动处理失败消息，提高系统可靠性

⏰

延时消息

支持延时消息和定时消息，提供灵活的时间控制策略
